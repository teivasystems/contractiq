# ContractIQ — ServiceNow Contract Intelligence

Interactive mockup for ContractIQ, a ServiceNow scoped application that turns raw contract and platform usage data into financial intelligence.

## Live Demo

**[https://teivasystems.github.io/contractiq/](https://teivasystems.github.io/contractiq/)**

## Local Development

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173/contractiq/`

## Deployment

Deploys automatically to GitHub Pages on every push to `main` via GitHub Actions.

### First-time setup

1. Go to **Settings → Pages** in this repo
2. Under **Build and deployment**, select **GitHub Actions** as the source
3. Push to `main` — the workflow handles the rest

## Tech Stack

- React 18
- Recharts (charts & data visualisation)
- Lucide React (icons)
- Vite (build tool)
